Main project - EStockData

AngularJS single-page site which let a regitered user:

  • Manage a virtual stock portfolio with charts and stock info
  • Set alerts to sell a stock with 'take-profit' or 'stop-loss'
  • See Cryptographic currencies historic prices and charts

The site uses:

  • RESTapi GET data from two external API's
  • RESTapi GET/POST/DELETE user data to/from the DB which sits on a far-end Heroku-JSon-Server
  • EMail Service to send the user alert triggered notice
